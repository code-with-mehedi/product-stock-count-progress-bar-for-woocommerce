This plugin will help you to display the woocommerce product stock quantity in a horizontal progress bar.

== Description ==
Just install and activate the plugin then go to your wocommerce product section and enable stock quantity for it.


== Installation ==
1. Download and unzip the plugin. Upload the unzipped folder to the wp-contents/plugins folder of your WordPress installation.
2. Active the plugin from the WordPress Plugins administration page.
3. OR, Got to WP admin panel, click 'Plugins' -> 'Add new'. In the search input box, type 'Woo stock count progress bar'.
4. Install and activate the plugin.
